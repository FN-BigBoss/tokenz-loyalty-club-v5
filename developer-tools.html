Quick Start Guide

Get up and running with TokenZ API in minutes.

1. Get Your API Key

First, you'll need to obtain your API key from the developer dashboard.

# Your API key
const apiKey = 'tk_live_1234567890abcdef';

2. Make Your First Request

Here's how to make a simple API call to get user information.

const response = await fetch('https://api.tokenz.com/v2/users/me', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  }
});
const user = await response.json();
console.log(user);

3. Handle the Response

All API responses follow a consistent format.

{
  "success": true,
  "data": {
    "id": "user_1234567890",
    "email": "developer@example.com",
    "name": "John Developer",
    "created_at": "2024-07-13T02:20:00Z"
  }
}
